# alias
Install the tool and its not working or running funny
Try using `which` to check how its installed.
E.g.
```bash
which gau
```
Sometimes is has an existing `alias`

See the alias for a command
```bash
alias gau
```

Remove the alias

```bash
unalias gau
```

Make permanent change
```bash
source ~/.zshrc 
```

Reload shell config
```bash
source ~/.zshrc 
```
